# Data Analytics Internship - Final Submission

This repository contains my completed tasks for the one-month Data Analytics Internship. I have completed **8 out of 9 tasks**, including all required bonus objectives for each task (excluding Task 8).

## Project Overview
- **Duration:** 1 Month
- **Tasks Completed:** 1, 2, 3, 4, 5, 6, 7, and 9.

## Task Summary & Key Achievements

### Task 1: Sales Performance Dashboard (Excel)
* **Goal:** Analyze Superstore sales data.
* **Bonus:** Added interactive **Slicers** for region/category and calculated **YoY (Year-over-Year)** growth.

### Task 2: Titanic EDA (Python)
* **Goal:** Clean and explore survival patterns.
* **Bonus:** Created **Heatmaps** and detailed bar plots to visualize survival correlations.

### Task 3: Customer Segmentation (RFM Analysis)
* **Goal:** Segment customers based on Recency, Frequency, and Monetary value.
* **Bonus:** Developed marketing strategies for each segment (e.g., "At Risk" vs. "Champions").

### Task 4: Kaggle Survey Analysis
* **Goal:** Explore trends in the Data Science industry.
* **Bonus:** Compared salary distributions across different education levels.

### Task 5: SQL Analysis (Chinook Database)
* **Goal:** Query a digital media store database.
* **Bonus:** Visualized monthly revenue performance using Python after extracting SQL data.

### Task 6: Web Scraping (Fake Jobs)
* **Goal:** Scrape job titles and locations using BeautifulSoup.
* **Bonus:** Analyzed the most frequent skills required by city.

### Task 7: Time Series Forecasting (Walmart)
* **Goal:** Analyze weekly sales trends.
* **Bonus:** Implemented **Exponential Smoothing** and **Rolling Mean** forecasting models.

### Task 9: Business Insights & Executive Report (Olist E-commerce)
* **Goal:** End-to-end analysis of the Brazilian E-commerce dataset.
* **Bonus:** Included an **Executive Summary** identifying key drivers of delivery delays (Package Volume vs. Shipping Costs).
